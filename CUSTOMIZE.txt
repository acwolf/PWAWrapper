### 1. GET STARTED (Required)
Open 'local.properties' and add:
app.url="https://your-website.com"
app.login="https://api.yourdomain.com/api/login"

Once finished, go to 'Build > Rebuild Project'.

---

### 2. UI CUSTOMIZATION
* **App Name:** 'res/values/strings.xml' -> 'app_name'.
* **Colors:** 'res/values/themes.xml' -> 'colorPrimary'.
* **App Icon:** Right-click 'res' -> New -> Image Asset.

---

### 3. HOW TO ADD A LOGO (Optional)
If you want to add a logo to the Login screen:
1. **Add File:** Put 'logo.png' in 'app/src/main/res/drawable'.
2. **Add View:** In 'activity_login.xml', add an <ImageView> at the top:

   <ImageView
       android:layout_width="150dp"
       android:layout_height="150dp"
       android:src="@drawable/logo"
       android:layout_marginBottom="24dp" />

---

### 4. OFFLINE HANDLING
The app uses a 'webview' component.
* To customize the offline screen, edit 'layout_offline.xml'.
* The app checks internet status via ConnectivityManager before loading.
